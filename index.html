Give UI feedback on live webapps

Get feedback on your work directly on the live site in an easy way.

For webdevelopers, webdesigners, ux designers and anyone who has ever received feedback from customers.

No need to inject a single script, works with closed and open source projects.

Code stays intact

No need to inject a single script tag.

Code can be automatically deployed from git (in dynamic docker containers) or you can specify an external live site.

Collaborative & centralised

You can see others' feedback and discuss in threads. No more isolated email conversations, all comments and survey answers in one place.

Focus & clarity

Surveys give the developer the possibility to ask specific questions and get specific answers easily.

Tagging elements

No more "that red button" feedback, nor sending screenshots, just tag the DOM element in the comment.

Deployment made so easy that it can't be true

1.1. Give us a git repository url and we will automatically clone the code from git and spin the app up in a docker container.

1.2. Or you could supply feedbacker with an external site url—e.g. Product Hunt

2. Finally you will get a shareable url that can be shared to testers, clients, customers or stakeholders.

Oh, and we support all tech stacks.

Want to try it out?

We will reach out to you and give you test access to feedbacker.site.
